Исследование задачи на чувствительность

Исследование задачи на чувствительность служит для изучения изменений оптимального решения и значения целевой функции при изменениях вектора правых частей ограничений и вектора коэффициентов целевой функции, а так же для изучения интервалов, которым должны принадлежать изменяемые значений.
После решения канонической задачи ЛП, вектор оптимального решения z можно разделить на два подвектора: zБ и zС, в первом из которых содержатся базисные компоненты, а во втором — нулевые свободные.
Соответственно, матрицу A можно разделить на две подматрицы — AБ и AС.
После этого оптимальное решение задачи можно представить следующим образом:

zБ = (AБ)-1·b


Изменяя значения компонентов вектора b, можно изучить, как при этом изменяются значения компонентов оптимального решения и значение целевой функции. Для того, чтобы решение оставалось допустимым, изменять значения компонентов вектора b следует в пределах интервалов устойчивости.
Для вектора c действия аналогичны, но рассматривается двойственная задача.
